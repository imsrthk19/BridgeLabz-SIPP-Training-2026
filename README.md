# Coding Standards Followed

## Program Naming Convention

Meaningful names are used for programs.

Examples:

✅ EmployeeWageCalculator.java  
✅ PalindromeChecker.java  
✅ StudentManagementSystem.java  


Avoid:

❌ Program1.java  
❌ Test.java  
❌ Demo.java
